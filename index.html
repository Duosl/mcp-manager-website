工作原理

简单三步，轻松管理您的 MCP 配置

1

添加配置

点击添加按钮，输入配置名称和 JSON 内容，系统会自动验证格式。

2

管理分类

使用标签系统对配置进行分类，通过搜索功能快速找到所需配置。

3

一键使用

点击复制按钮将配置复制到剪贴板，直接粘贴到 MCP 客户端中使用。
